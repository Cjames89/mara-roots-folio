# Connor Selfridge — Personal Portfolio

Personal portfolio site for Connor Selfridge, Revenue Operations specialist at MARA Holdings.

**Live site:** [connorselfridge.com](https://www.connorselfridge.com)

## Tech Stack

- **Vite** + **React** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **Framer Motion** for animations

## Local Development

```sh
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

Deployed to GitHub Pages. Push to `main` to trigger a new build.
